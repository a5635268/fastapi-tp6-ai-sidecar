## 功能概述

使用阿里云百炼 QwenVL-Max 视觉大模型，在文章爬取过程中自动进行：
1. **智能筛选**：判别图片是否为信息图（含图表、数据、长文本）
2. **数据提炼**：从信息图中提取结构化 Markdown 数据
3. **图片存储**：有价值的信息图自动上传到阿里云 OSS 保存

## 核心流程

```
文章爬取 (parse_article)
    ↓
HTML → Markdown 转换完成
    ↓
调用 VlmImageService.clean_and_extract()
    ↓
    ├─→ 正则提取所有图片 ![alt](url)
    ├─→ 并发下载图片（处理防盗链）
    ├─→ 长图切片（高度 > 2000px）
    ├─→ 并发调用 QwenVL-Max API
    │       ├─→ 返回 "SKIP" → 删除图片 Markdown
    │       └─→ 返回提炼文本 → 上传 OSS → 替换为文本块
    └─→ 返回清洗后的 Markdown
    ↓
返回 ParseResult（包含提炼后的 markdown）
```

## 配置说明

### 1. 配置 `.env` 文件

复制 `.env.example` 为 `.env` 并填写配置：

```bash
# Qwen-VL 配置
QWEN_API_KEY=sk-xxx  # 阿里云百炼 API Key
QWEN_MODEL=qwen-vl-max-latest

# 阿里云 OSS 配置
OSS_ENDPOINT=oss-cn-hangzhou.aliyuncs.com
OSS_BUCKET=your-bucket-name
OSS_ACCESS_KEY_ID=your-access-key-id
OSS_ACCESS_KEY_SECRET=your-secret
OSS_IMAGE_PREFIX=article-images/
```

### 2. 获取配置凭证

**Qwen API Key**：
1. 登录 [阿里云百炼控制台](https://bailian.console.aliyun.com/)
2. 进入 "API-KEY 管理" 页面
3. 创建 API Key 并复制

**OSS 凭证**：
1. 登录 [OSS 控制台](https://oss.console.aliyun.com/)
2. 进入目标 Bucket
3. 在 "设置" → "访问控制" 中获取密钥

## 技术特性

### 并发处理
- 使用 `asyncio.gather()` 并发调用 QwenVL API
- 信号量控制最大并发数（默认 10）
- 10 张图片处理耗时约 5-10 秒

### 长图切片
- 使用 Pillow 自动检测图片高度
- 高度 > 2000px 自动切分为多张
- 切片结果合并输出

### 防盗链处理
- 下载微信图片伪装 `Referer: https://mp.weixin.qq.com`
- 转 Base64 传给 QwenVL API

### 错误处理
- 单图失败不影响整体流程
- OSS 上传失败保留提炼文本
- QwenVL API 失败默认 SKIP

### 倒序替换
- 从后往前替换 Markdown
- 避免字符串索引偏移问题

## 使用示例

### 1. 爬取文章（自动触发 VLM 处理）

```bash
curl -X POST "http://localhost:8000/api/v1/article/parse" \
  -H "Content-Type: application/json" \
  -d '{"url": "https://mp.weixin.qq.com/s/xxx"}'
```

返回的 `markdown` 字段中：
- 无效图片（封面、二维码）已被删除
- 信息图位置替换为：
```markdown
> 📊 **【图表数据提炼】**：
> - 核心数据 1
> - 核心数据 2
> - 关键结论

> (原图：https://your-bucket.oss-cn-hangzhou.aliyuncs.com/article-images/xxx.jpg)
```

### 2. 手动调用（可选扩展）

可以在 `app/routers/vlm.py` 中添加独立 API（暂未实现）：
```python

async def process_images(request: VlmImageAnalysisRequest):
    vlm_service = get_vlm_service()
    markdown = await vlm_service.clean_and_extract(request.markdown)
    return ResponseBuilder.success(data={"markdown": markdown})
```

## 故障排查

### QWEN_API_KEY 未配置警告

```
[WARNING] [VLM] QWEN_API_KEY 未配置，图片处理功能将不可用
```

**解决**：在 `.env` 中设置 `QWEN_API_KEY` 并重启服务。

### OSS 上传失败

```
[WARNING] [OSS] 图片上传失败 error=...
```

**解决**：
1. 检查 OSS 配置项是否完整
2. 验证 OSS Bucket 权限
3. 检查网络连通性

### 图片下载失败

```
[WARNING] [VLM] 图片下载失败 url=... error=403
```

**解决**：
- 微信图片有防盗链，已在代码中处理
- 如仍失败，可能是图片链接过期或需要 Cookie

### 长图切片丢失细节

**解决**：调整切片阈值，修改 `VlmImageService._slice_long_image()` 中的 `max_height` 参数。
